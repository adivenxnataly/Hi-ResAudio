## Hi-Res Audio™
 Enable high resolution audio in audio_policy config file for MediaTek only up to 32-bit/192kHz (if device supports).
 
## Requirement
 this is module so install using Magisk app:

  - Android 12 (SDK 31)
  - MTK devices
  
 *does not cause softbrick or bootloop.

## Tested on (Stock ROM)
  - Android 12 - MIUI 13
  - Android 13 - MIUI 14
  - Android 14 - HyperOS 1.0
